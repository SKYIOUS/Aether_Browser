# Contributing to Aether Browser

We welcome contributions to the Aether engine and Korlang language.

## 📜 Guidelines

1.  **Originality**: All engine code must be original. Do not copy from other browser engines.
2.  **Safety**: Use Rust's safety features. Avoid `unsafe` unless absolutely necessary for FFI.
3.  **Korlang**: UI changes should be made in `.kor` files whenever possible.
4.  **Tests**: Ensure all unit tests pass before submitting.

## 🛠 Workflow

- Fork the repository.
- Create a feature branch.
- Submit a PR with a detailed description of changes.

Thank you for helping build the future of the web!
