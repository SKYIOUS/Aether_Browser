# Standards Compliance Roadmap

This document outlines the roadmap for ensuring Aether Browser's adherence to web standards (HTML5/CSS3).

## Current Status
- **Foundation:** Initial structure established in `src/engine/standards/`.
- **Parser:** Migration towards HTML5-compliant state machine tokenizer in progress.

## Supported Specification Subsets
- **HTML5:** Basic tokenization started.
- **CSS3:** Basic parsing structural foundation laid.

## Future Milestones
1. Complete HTML5 state machine tokenizer.
2. Implement HTML5 tree builder.
3. Expand CSS3 selector support.
